# Food Platform

Веб-платформа доставки еды: каталог товаров, корзина, оформление заказов, личный кабинет и админ-панель.

## Стек

- **Backend:** Django 6.x, Python 3.12+
- **Frontend:** Bootstrap 5, Material Symbols иконки, Flatpickr (библиотека выбор даты)
- **БД:** SQLite
- **Медиа:** Pillow (для изображения товаров)

## Структура проекта

```
food_platform/           # настройки проекта
  settings.py
  urls.py
  templates/admin/       # шаблоны смены пароля в админке

products/                # каталог, корзина, заказы, рейтинги
  models.py              # Product, ProductCategory, Basket, Order, OrderItem, ProductRating (модели проекта)
  views.py               # компоненты 
  forms.py               # форма оформления заказа
  context_processors.py  # итог по корзине для навбара
  templates/products/    # шаблоны

users/                   # регистрация, вход, профиль, смена пароля
  models.py              # кастомная модель User (профиль: телефон, дата рождения, пол)
  views.py               # компоненты
  forms.py               # формы
  templates/             # шаблоны

static/css/style.css     # стили (тёмная тема, карточки, уведомления, таблицы)
```

## Основные URL

| Путь                      | Описание                                      |
|---------------------------|-----------------------------------------------|
| `/`                       | Главная                                       |
| `/catalog/`               | Каталог товаров с рейтингом и корзиной        |
| `/basket/`                | Корзина                                       |
| `/create-order/`          | Оформление заказа (карта, адрес, комментарий) |
| `/users/login/`           | Вход                                          |
| `/users/register/`        | Регистрация                                   |
| `/users/profile/`         | Профиль (просмотр/редактирование)             |
| `/users/password-change/` | Смена пароля (пользовательский интерфейс)     |
| `/admin/`                 | Админка Django                                |
| `/admin/password_change/` | Смена пароля в стиле админки                  |

## Установка

1. **Распаковать архив** в удобную папку с помощью win-rar и открыть ее в командной строке

2. **Создать виртуальное окружение и активировать:**
   ```bash
   python -m venv .venv
   ```
   ```bash
   .venv\Scripts\activate
   ```

3. **Установить зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Запустить сервер:**
   ```bash
   python manage.py runserver 8001
   ```
   Открыть в браузере: http://127.0.0.1:8001/

## Функциональность

- Регистрация и вход, расширенный профиль (имя, фамилия, пол, дата рождения, телефон).
- Каталог товаров по категориям, рейтинг звёздами (авторизованные пользователи могут ставить оценку).
- Корзина: добавление, изменение количества, удаление, очистка; в навбаре - иконка корзины с суммой.
- Оформление заказа: номер карты (форматирование по 4 цифры), адрес доставки, комментарий.
- Тестовые уведомления (в правом верхнем углу под навбаром).
- Адаптивное меню (бургер и выезжающая панель на мобильных).
- Админка: русские названия моделей и приложений, смена пароля в оформлении админки.
